## 项目简介

这是一个专门为从 Vue 转向 React 的开发者设计的学习项目。通过循序渐进的示例和详细的注释，帮助你快速掌握 React 的核心概念和最佳实践。

## 特点

- 循序渐进的学习路径
- Vue 与 React 概念对比
- 详细的代码注释和说明
- 实用的示例和最佳实践
- TypeScript 支持

## 快速开始

1. 克隆项目
```bash
git clone [your-repository-url]
```

2. 安装依赖
```bash
cd react-learning
npm install
```

3. 启动开发服务器
```bash
npm start
```

4. 在浏览器中打开 http://localhost:3000
